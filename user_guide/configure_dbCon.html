Configure your application

Author : Judywen Guapin

  • 1. Inside of the root directory, find "jag.ini.php".

  • 2. LOAD DEFINES : basically defining the root access and system name.

    • ROUTE : define the url path of calling every pages.

    • SYSTEM_PATH : define the main path of your system.

    • SYSTEM_NAME : define the desired name of your application.

  • 3. SET SITE UNDER MAINTENANCE : basically defining to shutdown the site and display maintenance.

  • 4. SET DATABASE CONFIGURATIONS : basically defining the connection of your database, very basic.

  • 5. SET ENVIRONMENT : basically defining the mode of the system.

  • 6. VALUE : basically defining all the required core functions of the system.
